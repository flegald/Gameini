Where to find your game's .ini file

If you are editting a Steam game most Steam games live in
/programfiles/steam/app/common/
You should have a list of games in your inventory and you can search around those files.

Another very helpful way to find the location of a game's .ini file is to Google it.
e.g. "Civ 5 ini file location"
