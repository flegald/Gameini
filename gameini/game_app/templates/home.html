  • How To Use This Site

    Either upload one of your own .ini files to edit, or select one from the drop down menu.
    If you choose to upload your own make sure to put the ORIGINAL FILE'S name in the title box without the .ini extension.
    Once you have changed the fields you would like to change click 'download' and save the file where the game expects it to be.
    Replace the existing file

  • Where to find your game's .ini file

    If you are editting a Steam game most Steam games live in
    /programfiles/steam/app/common/
    You should have a list of games in your inventory and you can search around those files.

    Another very helpful way to find the location of a game's .ini file is to Google it.
    e.g. "Civ 5 ini file location"
